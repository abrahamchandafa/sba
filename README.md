# SBA DRAFT WEBSITE

Static website served using nginx in a Docker container.

## Project Structure
```
sba-draft-new/
├── public/                    # Static assets
│   ├── full_logo.png
│   ├── ico.png
│   ├── logo.png
│   └── transparent_logo.png
├── index.html                 # Main HTML file
├── styles.css                 # CSS styles
```

## Files
- `index.html` - Main webpage
- `styles.css` - Styling
- `public/` - Images and static assets
